# JavaScript Problem-Solving Practice

## Project Description

This repository contains solutions to coding challenges from Codewars and LeetCode, aimed at improving JavaScript skills and problem-solving abilities.

## Usage Instructions

1. Navigate to the directory of the challenge you want to review.
2. Each challenge directory contains a `.js` file with the solution.
3. To run the solution, use Node.js or any JavaScript runtime.
